NotTooExpensive

A lightweight economy-balancing plugin that keeps progression fair and server pricing under control.

README-based documentation page • April 10, 2026

Overview

NotTooExpensive helps server owners enforce sensible price limits across economy-related systems. It prevents accidental or abusive pricing spikes so players can progress naturally without economy lockouts.

Features

  • Configurable maximum and minimum pricing thresholds.
  • Protection against unrealistic economy values.
  • Simple setup with readable defaults.
  • Designed for low-overhead, production server usage.

Installation

  1. Download the latest NotTooExpensive.jar.
  2. Place the jar into your server's plugins/ folder.
  3. Restart the server to generate configuration files.
  4. Adjust values in the config and restart/reload.

Quick Configuration

Example configuration structure:

limits:
  minimum-price: 1
  maximum-price: 50000
  strict-mode: true

messages:
  warn-player: true
  log-adjustments: true

Commands

  • /nottooexpensive reload — reloads plugin configuration.
  • /nottooexpensive status — shows current active limits.

Permissions

  • nottooexpensive.admin — access admin subcommands.
  • nottooexpensive.reload — reload plugin config.
